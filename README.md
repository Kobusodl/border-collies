# GitHub Pages kennel site

This is a single-file static website template designed for a dog breeder or kennel.

## Files
- `index.html` — full website with inline CSS and JavaScript

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/(root)** folder.
6. Save.
7. GitHub will publish the site on a free `github.io` URL.

## To use a custom domain
1. Buy a domain from a registrar.
2. In GitHub Pages settings, add the custom domain.
3. Update your DNS records as instructed by GitHub.

## Notes
- This version is fully static, so it works well on GitHub Pages.
- For a working contact form on GitHub Pages, use a third-party form service like Formspree, Getform, or Basin.
- The current version uses placeholder content that should be replaced with the breeder's real text, phone number, email, WhatsApp number, and photos.
